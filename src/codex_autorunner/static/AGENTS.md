# Static Assets - Agent Guide

This directory holds the web UI shell and generated assets.

## Route Here When

- You are editing `index.html`, `styles.css`, vendor assets, or checking generated output.
- You need the rendered asset path after a frontend build.

## Keep Straight

- Edit TypeScript in `../static_src/`, not `generated/*.js`.
- Edit `index.html` and `styles.css` directly here.
- `assets.json` is generated by postbuild.
- Read `STYLE_GUIDE.md` only when the task touches layout, spacing, or visual patterns.

## Validation

- Run `pnpm run build` after TS changes.
- Run `make frontend-check` after HTML/CSS changes.
- If static assets fail to load or refresh, continue in `../surfaces/web/AGENTS.md`.
